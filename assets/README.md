# NearBuy Platform - Assets Directory

This directory is intended for storing static assets such as images, logos, and other media files.

Currently using placeholder images from `placehold.co` for demonstration purposes.

## Recommended Assets to Add

- Store logos/images
- Product images
- Brand logo
- Favicon

## Using Real Images

Replace placeholder URLs in `js/db.js` with relative paths to images in this directory:

```javascript
imageUrl: "./assets/stores/pizza-paradise.jpg"
```
